# 主窗口 Pin 置顶功能说明

## 目标

当你浏览其他应用时，可将 MdMonitor 主窗口置顶固定在前景，方便一边看链接收集结果一边操作其他窗口。

## 功能点

- 主窗口顶部提供 Pin 图标按钮：
  - 未置顶：`pin`
  - 已置顶：`pin.fill`
- 置顶后窗口保持可拖动，可在屏幕上自由定位。
- 置顶时可配置透明度（40% - 100%）。
- 置顶时可选开启鼠标穿透（点击会落到下层应用）。
- 菜单栏提供“取消置顶”入口，避免穿透开启后无法直接点击窗口。

## 使用方式

1. 打开主窗口，在预览页顶部点击 Pin 图标启用置顶。
2. 进入设置页调整：
   - `置顶透明度`
   - `置顶时允许鼠标穿透`
3. 若开启穿透后不便操作，可在菜单栏点击 `取消置顶` 恢复。

## 行为说明

- 仅 `Pin` 状态下，透明度和穿透会作为长期窗口状态生效。
- 进入设置页时会按当前已保存值立即应用透明度预览，便于调参观察。
- 设置页内修改透明度/穿透时会实时预览。
- Pin 开关状态会持久化，关闭主窗口后从 menubar 重新打开会沿用上次置顶状态。
- 离开设置页时：
  - 未保存：回滚到“已保存配置 + 当前置顶状态”对应样式。
  - 已保存：按“已保存配置 + 当前置顶状态”继续生效。
- 非置顶状态下离开设置后窗口恢复：
  - 普通窗口层级
  - 100% 不透明
  - 不穿透
